
  PythonStatSVN
  ==============

   PythonStatSVN(pss)是一个针对SVN代码仓库的统计分析工具，它从SVN log和svn diff生成
   HTML报告。
   
   他用python语言编写而成，需要python3.6及以上版本来运行。
   
   图形部分采用echart



  快速开始

      * 下载最新的代码放到c:\pss目录下。
      
      * 执行statsvn.py your_project  
      
      * 结果输出到当前目录下index.html中，打开即可。
